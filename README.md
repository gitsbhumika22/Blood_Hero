# 🩸 Blood Hero - Blood Donation & Request Management App

Blood Hero is a cross-platform Flutter application designed to simplify blood donation and emergency blood request management. The application connects blood donors, patients, and hospitals through a centralized digital platform, enabling faster communication during emergencies.

Built using **Flutter** and **Firebase**, the application provides secure authentication, donor management, blood request posting, appointment booking, and donation history tracking.

#  Project Overview

Blood shortages during emergencies often occur due to the lack of an efficient communication system between blood donors and hospitals.

Blood Hero provides a centralized solution where users can:

- Register as blood donors
- Search for blood donors
- Request blood during emergencies
- Book donation appointments
- Receive notifications for blood requests
- Maintain donation history
- Manage user profiles

The application helps reduce response time and improves coordination between donors and recipients.

---

# ❗ Problem Statement

During medical emergencies, patients and hospitals often struggle to find eligible blood donors quickly.

Traditional blood donation systems rely on phone calls and manual records, leading to:

- Delay in locating donors
- Poor communication
- Unorganized donor records
- No real-time updates
- Lack of centralized management

Blood Hero addresses these challenges using a modern mobile application built with Flutter and Firebase.
